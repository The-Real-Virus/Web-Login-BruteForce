This tool is designed to run on Kali Linux.

# Dependencies
You need to install the following Python libraries:

'requests' to work with the web login.


# Installation
NOTE=> in Kali Linux These are Pre-installed. If not, Then:

open terminal and give below commands :

Step 1: Install Python 3 and pip (if not already installed)

sudo apt update
sudo apt install python3 python3-pip

Step 2: install requests module.

sudo apt update
sudo apt install python3-requests

( Now close the terminal and open new one to run the tool )
